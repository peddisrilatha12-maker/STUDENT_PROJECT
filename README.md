# Campus Records

Campus Records is a React and Vite project for viewing student information through multiple routed pages.

## How To Run

Install Node.js first. Then check the versions:

```bash
node -v
npm -v
```

Open this project folder in a terminal and install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the local URL shown in the terminal. It is usually:

```text
http://localhost:5173/
```

## Commands

Build the project:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Run lint checks:

```bash
npm run lint
```

## Project Notes

The student records are stored locally in:

```text
src/data/learners.js
```

This project demonstrates React Router pages, navigation, dynamic student profile routes, reusable components, and responsive styling.
